# smsSender
Simple SMS Sender App using the TWILIO API Gateway

## Requirements
- Windows OS
- Python 3.x
- Twilio ACC (Account SID, Authentication Code, Phone Number)
### Required Python3 Moules
- os, sys, threading
- json
- twilio
